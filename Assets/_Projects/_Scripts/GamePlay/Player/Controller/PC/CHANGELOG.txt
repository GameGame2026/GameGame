【Animator参数（共5个）】

| 参数名      | 类型    | 默认值 | 说明                     |
|-------------|---------|--------|--------------------------|
| Speed       | Float   | 0      | 移动速度（0/1/2）        |
| Grounded    | Bool    | true   | 是否在地面               |
| Jump        | Trigger | -      | 跳跃触发                 |
| FacingAway  | Bool    | false  | 是否背对摄像机           |
| Points      | Int     | 0      | 当前等级（0-3）          |

【动画文件命名规范】

格式：{Action}_{Direction}_{Level}.anim

示例：
- Idle_Front_0.anim（0级面对摄像机的Idle）
- Walk_Back_2.anim（2级背对摄像机的Walk）
- Jump_Front_3.anim（3级面对摄像机的Jump）

【控制逻辑】

输入 -> 动画状态：
- 无输入 -> Idle（根据记忆Front/Back）
- W键 -> Back动画组（背对）
- S键 -> Front动画组（面对）
- A/D键 -> Front动画组 + Scale翻转
- Shift -> 切换Walk/Run
- Space -> Jump
- 点数变化 -> 切换动画集（0-3）
