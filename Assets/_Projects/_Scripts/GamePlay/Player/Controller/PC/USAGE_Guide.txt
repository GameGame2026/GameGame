多等级2D角色动画系统 - 完整使用指南
====================================================================

【系统总览】

你的角色动画系统现在支持：
✅ 4个等级（0-3点）- 随游戏进度改变角色外观
✅ 2个朝向（Front/Back）- 按W背对，按S面对
✅ 左右镜像翻转 - 通过Scale.x实现
✅ 4个基础动作（Idle, Walk, Run, Jump）
✅ Shift加速功能
✅ 方向记忆（Idle时保持最后的朝向）

总计需要：4等级 × 2朝向 × 4动作 = 32个动画文件

【文件清单】

等级0（0点）：
  Idle_Front_0.anim, Walk_Front_0.anim, Run_Front_0.anim, Jump_Front_0.anim
  Idle_Back_0.anim, Walk_Back_0.anim, Run_Back_0.anim, Jump_Back_0.anim

等级1（1点）：
  Idle_Front_1.anim, Walk_Front_1.anim, Run_Front_1.anim, Jump_Front_1.anim
  Idle_Back_1.anim, Walk_Back_1.anim, Run_Back_1.anim, Jump_Back_1.anim

等级2（2点）：
  Idle_Front_2.anim, Walk_Front_2.anim, Run_Front_2.anim, Jump_Front_2.anim
  Idle_Back_2.anim, Walk_Back_2.anim, Run_Back_2.anim, Jump_Back_2.anim

等级3（3点）：
  Idle_Front_3.anim, Walk_Front_3.anim, Run_Front_3.anim, Jump_Front_3.anim
  Idle_Back_3.anim, Walk_Back_3.anim, Run_Back_3.anim, Jump_Back_3.anim

【组件配置】

1. Player GameObject需要以下组件：
   ✅ CharacterController
   ✅ PlayerInputHandler
   ✅ ThirdPersonController
   ✅ PlayerStats（新增）
   ✅ Animator
   ✅ PointsTester（可选，用于测试）

2. PlayerStats组件设置：
   - Max Points: 3
   - Current Points: 0（起始点数）
   - 其他属性保持默认

3. Animator设置：
   - 参考 Animator_MultiLevel_Guide.txt 进行详细配置
   - 需要添加5个参数：Speed, Grounded, Jump, FacingAway, Points

【代码接口】

// 获取PlayerStats组件
PlayerStats playerStats = player.GetComponent<PlayerStats>();

// 增加点数（例如：玩家完成任务）
playerStats.AddPoints(1);

// 减少点数（例如：玩家受到惩罚）
playerStats.RemovePoints(1);

// 直接设置点数（例如：加载存档）
playerStats.SetPoints(2);

// 获取当前点数
int currentPoints = playerStats.Points;

// 获取最大点数
int maxPoints = playerStats.MaxPoints;

【测试流程】

步骤1：基础测试
--------------------------
1. 启动游戏
2. 在Player上添加PointsTester组件（如果还没有）
3. 按数字键0-3快速切换等级
4. 观察角色动画是否正确切换

步骤2：动画测试矩阵
--------------------------
对每个等级（0-3）测试以下情况：

□ Idle_Front: 站立不动（面对摄像机）
□ Idle_Back: 按W移动后停止（背对摄像机）
□ Walk_Front: 按S或A/D移动
□ Walk_Back: 按W移动
□ Run_Front: 按Shift+S或Shift+A/D
□ Run_Back: 按Shift+W
□ Jump_Front: 面对摄像机时按Space
□ Jump_Back: 背对摄像机时按Space
□ 左右翻转: 按A和D时角色镜像翻转

步骤3：切换测试
--------------------------
1. 在运行中按0-3切换等级
2. 确认动画瞬间切换且不卡顿
3. 检查Console是否有以下日志：
   - "[PlayerStats] 点数设置: X -> Y"
   - "[ThirdPersonController] 动画点数更新: Y"

【快捷键说明】

游戏内控制：
- WASD: 移动
- Shift: 冲刺
- Space: 跳跃
- A/D: 左右翻转

测试控制（需要PointsTester组件）：
- 0/1/2/3: 设置点数为 0/1/2/3
- +（等号键）: 点数+1
- -（减号键）: 点数-1

【Animator参数说明】

| 参数名      | 类型    | 默认值 | 说明                           |
|-------------|---------|--------|--------------------------------|
| Speed       | Float   | 0      | 0=Idle, 1=Walk, 2=Run          |
| Grounded    | Bool    | true   | true=地面, false=空中          |
| Jump        | Trigger | -      | 起跳��间触发                   |
| FacingAway  | Bool    | false  | false=面对, true=背对          |
| Points      | Int     | 0      | 0-3，控制使用哪套动画集        |

【常见问题排查】

问题1：点数改变了但动画没切换
--------------------------
检查项：
□ PlayerStats组件是否在Player上
□ Animator是否配置了Points参数（Int类型）
□ Sub-State Machine之间的转换条件是否设置Points
□ Console是否有"动画点数更新"日志

问题2：某些动画丢失或播放错误
--------------------------
检查项：
□ 是否所有32个动画文件都已导入
□ 动画文件命名是否严格按照规范
□ Animator中是否正确引用了对应的动画
□ Blend Tree中的Motion是否设置正确

问题3：切换等级时有卡顿
--------------------------
解决方案：
□ 确保所有等级切换的Transition Duration = 0
□ 取消所有等级切换的Has Exit Time
□ 检查是否有循环转换导致的问题

问题4：左右翻转方向反了
--------------------------
解决方案：
在ThirdPersonController.cs的Move()方法中找到：
localScale.x = _isFacingRight ? Mathf.Abs(localScale.x) : -Mathf.Abs(localScale.x);

改为：
localScale.x = _isFacingRight ? -Mathf.Abs(localScale.x) : Mathf.Abs(localScale.x);

【性能优化建议】

1. 动画压缩：
   - 在Animation Import Settings中启用压缩
   - 对于简单的2D动画，可以降低关键帧数

2. Animator优化：
   - 使用Blend Tree而不是大量独立转换
   - 避免不必要的Transition（减少Any State使用）
   - 合理设置Transition Duration

3. 运行时优化：
   - 不要频繁修改Points值
   - 只在必要时触发点数变化事件

【进阶使用】

自定义点数获取逻辑：
```csharp
public class QuestReward : MonoBehaviour
{
    public void OnQuestComplete()
    {
        PlayerStats playerStats = FindObjectOfType<PlayerStats>();
        if (playerStats != null)
        {
            playerStats.AddPoints(1);
            Debug.Log("任务完成！获得1点成长值！");
        }
    }
}
```

保存/加载点数：
```csharp
public class SaveSystem
{
    public void SavePlayerData(PlayerStats stats)
    {
        PlayerPrefs.SetInt("PlayerPoints", stats.Points);
    }
    
    public void LoadPlayerData(PlayerStats stats)
    {
        int savedPoints = PlayerPrefs.GetInt("PlayerPoints", 0);
        stats.SetPoints(savedPoints);
    }
}
```
