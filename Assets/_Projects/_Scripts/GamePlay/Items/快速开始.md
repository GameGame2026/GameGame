# 收集系统 - 快速开始

1. **Collectible.cs** - 收集物核心脚本（叹号/问号物品）
2. **CollectibleManager.cs** - 收集管理器（单例，管理统计）
3. **CollectibleUI.cs** - 游戏中实时显示收集数量
4. **LevelSummaryPanel.cs** - 关卡结算界面
5. **LevelEndTrigger.cs** - 关卡结束触发器
6. **PlayerStats.cs** - 已添加 `SetAttackDamage()` 方法

## 🎯 核心功能

### 收集物特性
- ✅ 悬浮旋转动画
- ✅ 玩家靠近自动收集（默认2米范围）
- ✅ 播放音效
- ✅ 可选粒子特效
- ✅ UI实时更新

### 两种类型
- **叹号（⚡）AttackBoost** - 提升攻击力 1.25%/个
- **问号（❤）HealthBoost** - 提升最大生命 1.25%/个

### 结算系统
- 关卡结束显示收集统计
- 显示属性提升数值
- 自动应用到玩家属性
- 点击继续进入下一关

## 📝 5分钟快速设置

### 1. 创建收集物预制体（2分钟）

**叹号收集物：**
```
1. 层级面板右键 → 3D Object → Cube，命名 "Collectible_Attack"
2. 缩放到 (0.5, 0.5, 0.5)
3. 添加组件：
   - Collectible 脚本
   - Sphere Collider (勾选 Is Trigger, Radius=2)
   - Rigidbody (勾选 Is Kinematic)
4. Inspector 设置：
   - Collectible Type: AttackBoost
   - Collect Range: 2
   - Collect Sound: 拖入音效文件
5. 创建子物体显示"！"符号或使用材质颜色（如红色）
6. 拖到 Project 保存为预制体
```

**问号收集物：**
```
1. 复制 Collectible_Attack，重命名 "Collectible_Health"
2. Collectible 脚本改为: HealthBoost
3. 改为不同颜色（如绿色）
4. 保存为预制体
```

### 2. 设置场景管理器（1分钟）

```
1. 层级面板右键 → Create Empty，命名 "CollectibleManager"
2. 添加 CollectibleManager 脚本
3. Boost Multiplier: 0.0125 (已默认设置)
```

### 3. 创建游戏UI（1分钟）

在现有 Canvas 下：
```
1. 右键 Canvas → UI → Text - TextMeshPro，命名 "AttackBoostText"
   - 文本内容: "⚡ ×0"
   - 位置: 屏幕右上角
   
2. 复制一个，命名 "HealthBoostText"
   - 文本内容: "❤ ×0"
   - 位置: 攻击文本下方

3. 在 Canvas 下创建空物体 "CollectibleUI"
4. 添加 CollectibleUI 脚本
5. 连接两个文本引用

6. 回到 CollectibleManager，将 CollectibleUI 拖到 Collectible UI 字段
```

### 4. 创建结算界面（1分钟）

**简化版结算面板：**
```
1. Canvas 右键 → UI → Panel，命名 "LevelSummaryPanel"
2. 在 Panel 下创建文本（都用 TextMeshPro）：
   - "AttackBoostCountText" (显示收集数量)
   - "HealthBoostCountText" (显示收集数量)
   - "AttackIncreaseText" (显示提升数值)
   - "HealthIncreaseText" (显示提升数值)
   - "NewAttackText" (显示当前攻击)
   - "NewHealthText" (显示当前生命)
   - Button (命名 "ContinueButton"，文本改为"继续")

3. 在 LevelSummaryPanel 添加 LevelSummaryPanel 脚本
4. 连接所有6个文本和1个按钮
5. Next Scene Name: 输入下一关场景名（如 "Scene2"）
6. 取消勾选 LevelSummaryPanel 物体（默认隐藏）
```

### 5. 设置关卡结束触发器（30秒）

```
1. 在终点位置创建空物体 "LevelEndTrigger"
2. 添加 Box Collider (勾选 Is Trigger)
3. 添加 LevelEndTrigger 脚本
4. 拖入 LevelSummaryPanel 到 Summary Panel 字段
5. 调整 Collider 大小覆盖终点区域
```

### 6. 在场景放置收集物（30秒）

```
从 Project 拖拽预制体到场景中，随意摆放位置
建议每关 3-10 个
```

**收集3个问号：**
- 生命: 100 → 103.75 (+3.75)

**收集5个叹号 + 5个问号：**
- 攻击: 10 → 10.625 (+0.625)
- 生命: 100 → 106.25 (+6.25)

## 🔍 调试

如果不工作，检查：

1. **Player 标签** - 确保玩家物体标签是 "Player"
2. **PlayerStats** - 确保玩家有 PlayerStats 组件
3. **CollectibleManager** - 每个关卡场景都要有
4. **Collider** - 收集物的 Collider 勾选 Is Trigger
5. **Console** - 查看调试日志
